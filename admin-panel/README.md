# Lustril Admin Panel

Modern admin panel for managing the Lustril salon booking platform.

## Features

- 🔐 Secure admin authentication
- 💈 Salon approval/rejection system
- 👥 User management
- 📅 Booking overview
- 📊 Dashboard with statistics
- 🎨 Beautiful UI with Tailwind CSS

## Installation

```bash
cd admin-panel
npm install
```

## Development

```bash
npm run dev
```

The admin panel will be available at `http://localhost:3000`

## Build for Production

```bash
npm run build
```

## Default Admin Credentials

Create an admin user in the database first, then use those credentials to login.

## Pages

- **Dashboard**: Overview of platform statistics
- **Salons**: Approve/reject salon registrations
- **Users**: View and manage all users
- **Bookings**: Monitor all bookings
- **Analytics**: Platform insights (coming soon)

## Tech Stack

- React 18
- Vite
- Tailwind CSS
- React Router
- Axios
